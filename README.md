## Currently using

    - Arch Linux
    - Hyprland
    - swww for wallpapers
    - Hyprshot for screenshots
    - Waybar
    - Wofi
    - Zsh as shell
        - Oh my zsh
        - zoxide
        - zsh-autosuggestions
        - zsh-syntax-highlighting
    - Alacritty for terminal emulation
    - Nemo for file manager
    - NvChad + Neovim for text editing
